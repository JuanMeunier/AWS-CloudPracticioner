AWS WaveLength para Cloud Practitioner
¿Qué es WaveLength?
Infrastructure deployments embedded dentro de datacenters de telecom providers en el edge de 5G networks
Palabra clave para el examen:
"5G" en pregunta → Probablemente WaveLength
¿Qué hace WaveLength?
Despliega AWS services directamente en el edge de 5G networks
Ultra-low latency para aplicaciones 5G
Servicios que puedes desplegar en WaveLength Zones:

EC2 instances
EBS volumes
VPC

Arquitectura:
5G Mobile Device (usuario)
        ↓ ultra-low latency
WaveLength Zone (en 5G network edge)
- EC2 instances
- EBS volumes
        ↓ si necesario
Carrier Gateway
        ↓ secure connection
Parent AWS Region
- RDS
- DynamoDB
Características clave:
1. Ultra-Low Latency
Aplicación desplegada en el edge de 5G network
Usuario 5G → WaveLength Zone (muy cerca, latencia mínima)
2. Traffic Flow
Tráfico NO sale de CSP (Communication Service Provider) network
NO necesita llegar a AWS (pero puede si necesitas)
3. Connection to Parent Region
WaveLength Zone conectada a parent AWS Region
Use case: EC2 en WaveLength necesita acceder RDS/DynamoDB en región
4. No Additional Charges
Sin cargos extra por usar WaveLength
No service agreements adicionales
Casos de uso:
1. Smart Cities
Aplicaciones IoT urbanas con latencia mínima
2. ML-Assisted Diagnostics
Diagnósticos médicos en tiempo real
3. Connected Vehicles
Autos conectados requieren respuesta inmediata
4. Interactive Live Video Streams
Streaming con latencia ultra-baja
5. AR/VR (Augmented/Virtual Reality)
Experiencias inmersivas requieren latencia mínima
6. Real-time Gaming
Gaming móvil con respuesta instantánea
Tema común: Ultra-low latency + 5G + Edge computing

Resumen:
AWS WaveLength despliega AWS infrastructure (EC2, EBS, VPC) en edge de 5G networks dentro de telecom provider datacenters. Ultra-low latency para mobile users en 5G. Tráfico permanece en CSP network. 
Conectado a parent AWS Region. Sin costos extra. Casos de uso: Smart Cities, AR/VR, real-time gaming, connected vehicles, live streaming. En examen: 5G + ultra-low latency = WaveLength.
