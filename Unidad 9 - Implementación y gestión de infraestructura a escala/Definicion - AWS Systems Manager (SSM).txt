AWS Systems Manager (SSM) para Cloud Practitioner
¿Qué es SSM?
Servicio para gestionar fleets de EC2 instances y sistemas on-premises a escala
Tipo de servicio:
Hybrid AWS service - Funciona con:

EC2 instances en AWS
On-premises systems

Funcionalidades principales:
1. Operational Insights
Obtener estado de tu infraestructura
2. Suite de 10+ productos
No necesitas conocer todos para el examen
3 Features más importantes para el examen:
1. Automated Patching
Parchar (patch) automáticamente todos tus servers/instances
Enhanced compliance - Mantener sistemas actualizados
2. Run Commands
Ejecutar comandos across toda tu fleet de servers
Consistentemente en todos los servers a la vez
3. SSM Parameter Store
Almacenar parameter configuration
Centralized configuration management
Plataformas soportadas:

Linux
Windows
Mac OS
Raspberry Pi

¿Cómo funciona SSM?
Requisito: SSM Agent
SSM Agent = Programa pequeño que corre en background
Debe instalarse en:

EC2 instances
On-premises servers/VMs

Pre-instalado por defecto en:

Amazon Linux AMI
Algunos Ubuntu AMIs

Arquitectura:
EC2 Instances + On-Premises VMs
   (con SSM Agent instalado)
            ↓ report back
      SSM Service (AWS)
            ↓ send commands
EC2 Instances + On-Premises VMs
   (ejecutan commands, patches, etc.)
SSM Agent reporta a SSM Service
SSM Service envía commands/patches/config
Capabilities:
Patch servers - Todos a la vez
Run commands - Across entire fleet
Configure consistently - Misma configuración en todos

Troubleshooting:
Si instance no puede ser controlada por SSM:
Problema probablemente es SSM Agent:

No instalado
No funcionando correctamente
Sin permisos IAM

Qué es: Almacenamiento seguro para configuration data
Store:

Centralized y secure
Resumen:
SSM (Systems Manager) es hybrid service para gestionar fleets de EC2 instances y on-premises systems a escala. Funcionalidades clave: automated patching, run commands across fleet, Parameter Store. 
Requiere SSM Agent instalado en servers. 
En examen: patch fleet, run commands at scale, o hybrid management = SSM.
